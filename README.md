# Movie Recommendation System

A content-based Movie Recommendation System built with Streamlit that suggests movies based on similarity and user preferences.

## Overview
This project is designed to help users discover movies that match their interests. It uses machine learning techniques to analyze movie data and recommend similar movies efficiently.

## Features
- Search movies by name  
- Get personalized movie recommendations  
- Display movie posters using an external API  
- Recommendations based on similarity algorithms  
- Fast and responsive user interface  

## Tech Stack
- Framework: Streamlit  
- Language: Python  
- Libraries: Pandas, NumPy, Scikit-learn  
- API: TMDB (The Movie Database)

## How It Works
The system uses a content-based filtering approach. It analyzes movie attributes such as genres, keywords, cast, and overview, then computes similarity between movies to recommend the most relevant options.
